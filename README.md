# Car-Ecommerce
RecursionCS Vue.jsコースの課題です。

# Description
自動車販売ECサイトです。左上に2つセレクトメニューがあり一番左がブランドカテゴリーがあります。好きなブランドを選択するとそのブランドのみが表示されます。次はソートメニューでこれは価格順と日時が新しい順のメニューです。
